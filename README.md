# Overview

The **Multi Tenancy Addons** Solution Pack is for Managed Security Service Providers (MSSPs). This solution pack helps users experience the power and capability of FortiSOARâ„¢ incident response in a multi-tenant architecture and provides a template of mitigation playbooks that are agnostic of underneath tenants.

You can configure these remotely executable playbooks to execute mitigation actions on the tenants from the master node. For example, if you require to block a domain on a tenant node from the master node, then you can use the remote executable playbooks that are part of this solution pack to achieve this action.
